# LifeStream

> **Private, ad-free social networking for friends and family.**

LifeStream is a self-hosted social networking platform for privately sharing photos, stories, announcements, conversations, and life's milestones with the people who matter most.

Unlike traditional social networks, LifeStream is designed to strengthen existing relationships—not maximize engagement.

## Why LifeStream?

Most social media platforms are built around advertising, algorithms, followers, and engagement metrics.

LifeStream takes a different approach.

- 📷 Share photos, stories, and videos.
- 💬 Have meaningful conversations.
- ❤️ React without chasing likes.
- 👨‍👩‍👧‍👦 Stay connected with friends and family.
- 🔒 Keep your data private.
- 🚫 No ads. No tracking. No algorithms.

LifeStream values relationships over reach.

## Key Features

- Chronological feed
- Photos, stories, announcements, polls, and life events
- Rich conversations and reactions
- Fine-grained privacy controls
- Browser-first experience
- Progressive Web App (PWA)
- AI-assisted tagging and life-event suggestions
- Comprehensive automation API
- Self-hosted and privacy-focused

## Design Principles

LifeStream is guided by a few simple principles.

- Privacy first
- Conversation over consumption
- Simple by default
- Browser first
- User control
- AI as an assistant
- Personal over prescriptive
